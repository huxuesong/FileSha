Usage
-----

Usage of FileSha:

  -filters string
  
    	the files or directotys that will be ignored
      
  -resultDir string
  
    	the directory where will  store the result file (default "./")
      
  -resultFileName string
  
    	the result file name (default "result.txt")
      
  -root string
  
    	choose the directoty (default "./")
